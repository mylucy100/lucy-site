OpenClaw 的强大之处在于技能系统。截止 2026 年 3 月，ClawHub 上已有超过 13,000 个社区技能——但坦白说，其中 80% 要么质量一般，要么存在安全隐患。

如何在海量技能中找到真正有用的？我结合实际使用经验和社区反馈，精选了 20 个最值得安装的 OpenClaw 技能。

## 开发效率类

### 1. GitHub

**安装**：`clawhub install openclaw/github`

开发者必备技能。让你的 Agent 拥有完整的 GitHub 操作能力：

- 克隆仓库、创建分支、切换分支
- 提交代码、编写 commit message、推送
- 创建 PR、添加描述、审查代码
- 搜索 issue、创建新 issue

不用在终端和浏览器之间切换，用自然语言就能完成整个 Git 工作流。

### 2. GOG (Google Workspace)

**安装**：`clawhub install openclaw/gog`

ClawHub 下载量最高的技能之一（14K+）。统一管理你的 Google 服务：

- Gmail：搜索、撰写、回复邮件
- Google Calendar：查看日程、创建事件
- Google Drive：搜索、上传、下载文件
- Google Sheets/Docs：读写文档数据

### 3. Vercel

**安装**：`npx skills add vercel/skills`

用自然语言控制 Vercel 部署：

- 部署项目、回滚版本
- 查看部署状态和日志
- 管理环境变量

### 4. Composio

**安装**：`npx skills add composiohq/skills`

一个技能解锁 860+ 外部工具。GitHub、Slack、Gmail、Notion……全部通过统一接口访问，无需单独处理认证。

## 搜索研究类

### 5. Tavily Search

**安装**：`clawhub install framix-team/openclaw-tavily`

专为 AI Agent 设计的搜索引擎。返回结构化摘要，不是一堆链接：

- `tavily_search`：AI 优化的网页搜索
- `tavily_extract`：提取 URL 内容
- `tavily_crawl`：爬取网站数据
- `tavily_map`：映射网站结构

### 6. Exa Search

**安装**：`npx skills add exa-labs/skills`

开发者专用搜索引擎，索引来源包括官方文档、GitHub 仓库、编程论坛。搜索技术信息比通用引擎更精准。

### 7. Multi Search Engine

**安装**：`npx skills add openclaw/multi-search-engine`

集成 17 个搜索引擎（8 个国内 + 9 个国际），支持高级搜索语法、时间过滤、站内搜索，无需 API Key。

## 自动化工作流

### 8. N8N Workflow

**安装**：`clawhub install openclaw/n8n`

用对话控制你的 n8n 自动化平台，触发复杂工作流，无需在仪表盘操作。

### 9. Clawflows

**安装**：`clawhub install openclaw/clawflows`

多步骤工作流编排。把复杂任务拆解成可复用的流程，一键执行。

### 10. Mission Control

**安装**：`clawhub install openclaw/mission-control`

你的个人"控制塔"。每天早上自动聚合：

- 待办事项和截止日期
- 今日/明日日程
- 重要通知摘要
- AI 优先级排序

### 11. Capability Evolver

**安装**：`clawhub install capability-evolver`

ClawHub 下载量第一（35K+）。让 Agent 自我进化：

- 分析历史交互记录
- 识别高频任务模式
- 自动优化响应策略

适合长期运行的 Agent。

## 媒体与内容

### 12. PixVerse Skills

**安装**：`npx skills add pixverseai/skills --skill pixverse-ai-image-and-video-generator`

AI 媒体生成神器。支持 6 个视频模型和 7 个图像模型：

- 文字生成视频（1-10 秒，最高 1080p）
- 图片转视频
- 视频后处理：延长、放大、添加音效
- 批量生成

### 13. Eleven Labs Agent

**安装**：`clawhub install openclaw/elevenlabs`

给你的 Agent 加上"声音"。语音合成 + 电话回退——如果邮件或短信联系不上，直接打电话。

### 14. Summarize

**安装**：`clawhub install openclaw/summarize`

智能内容摘要。支持网页、PDF、图片、音频、YouTube 视频。

### 15. Markdown Converter

**安装**：`npx skills add openclaw/markdown-converter`

文档格式转换神器。支持 PDF、Word、Excel、PPT、HTML、图片、音频等格式转 Markdown。

## 智能与记忆

### 16. Self-Improving Agent

**安装**：`npx skills add charon-fan/agent-playbook --skill self-improving-agent`

让 Agent 记录错误、学习经验、优化偏好。越用越聪明。

### 17. Smart Agent Memory

**安装**：`clawhub install smart-agent-memory`

分层上下文记忆系统。按标签、技能、时间、实体类型加载记忆，让 Agent 拥有长期记忆能力。

### 18. Reverse Engineering

**安装**：`npx skills add wshobson/agents/protocol-reverse-engineering`

逆向工程利器。捕获网络流量、分析协议、生成 Wireshark 解析器和 Python 解析器。

## 实用工具

### 19. Weather

**安装**：`npx skills add openclaw/weather`

天气查询技能。通过 wttr.in 或 Open-Meteo 获取天气和预报，无需 API Key。

### 20. Home Assistant

**安装**：`npx skills add home-assistant/skills`

用自然语言控制智能家居。完全本地运行，零云端依赖。

## 安装技巧

### 推荐安装方式

**方式一：skills.sh（推荐）**

Vercel 官方维护的精选技能库，质量有保障：

```bash
cd ~/.openclaw
npx skills add <skill-url>
```

**方式二：ClawHub**

官方技能市场，数量最多：

```bash
clawhub install <skill-name>
```

**⚠️ 安全提示**：安装前务必检查 ClawHub 页面上的 Security Scan 状态，只安装标记为 Benign 的技能。

### 安装后验证

```bash
openclaw status
```

查看 Skills 部分，确认技能已激活。
