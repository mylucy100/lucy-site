# OpenClaw版本发布历史与重要更新

> "这不是一次普通的版本更新。108 次代码提交、9 项破坏性变更、Plugin SDK 全面重构、ClawHub 技能市场正式上线。"
> — 版本更新说明

## 6.1 早期版本（2025.11 - 2026.01）

**2025年11月：原点**

一切始于 2025 年 11 月的那个夜晚。Peter 只用了一个小时，就将 WhatsApp 和 Claude Code 连接起来，创建了 Clawdbot 的第一个原型。

这个时期的特点是：
- 极简的原型，功能有限
- 主要在 Peter 的朋友圈中传播
- 没有任何正式发布

**2026年1月25日：正式诞生**

Clawdbot/Moltbot 正式发布。

发布后 24 小时内获得 9000 颗星标，创造了 GitHub 历史上的记录。

这一时期的版本特性：
- 基础的消息收发功能
- 简单的文件操作
- 与 Claude API 的集成

**2026年2月：爆发式增长**

|------|----------|----------|

## 6.3 成熟期（2026.03）：v2026.3.22 重大更新

**2026年3月22日：里程碑版本**

这是一个具有里程碑意义的版本：

- **108 次代码提交**

3. **破坏性变更**

   - API 端点变化
   - 权限模型改进

4. **性能优化**
   - 内存使用降低 30%
   - 启动时间缩短 50%
   - 并发处理能力提升

## 6.4 Plugin SDK 重构

Plugin SDK 的重构是 v2026.3.22 最重要的技术变更之一。

**旧版 SDK 的问题：**

- API 设计不够直观
- 类型定义不完整
- 难以扩展

**新版 SDK 的改进：**

```typescript
// 新版 Plugin SDK 示例
import { Plugin, Tool, Context } from '@openclaw/plugin-sdk';

class MyPlugin implements Plugin {
  name = 'my-plugin';
  version = '1.0.0';

  tools: Tool[] = [
    {
      name: 'custom_tool',
      description: '一个自定义工具',
      parameters: {
        type: 'object',
        properties: {
          input: { type: 'string' }
        },
        required: ['input']
      },
      execute: async (params: { input: string }, context: Context) => {
        // 工具逻辑
        return { result: `处理: ${params.input}` };
      }
    }
  ];

  async onLoad() {
    console.log('插件加载');
  }

  async onUnload() {
    console.log('插件卸载');
  }
}
```

**新架构的优势：**

1. **类型安全**：完整的 TypeScript 类型定义
2. **声明式配置**：更清晰的配置方式
3. **生命周期管理**：明确的加载/卸载钩子
4. **工具注册**：统一的工具注册机制

## 6.5 ClawHub 技能市场上线

**ClawHub** 是 OpenClaw 官方推出的技能和插件市场。它的上线标志着 OpenClaw 生态系统的成熟。

**ClawHub 的功能：**

1. **技能发现**
   - 浏览热门技能
   - 搜索特定功能
   - 查看技能详情和评价

2. **一键安装**
   - 点击安装即可使用
   - 自动配置依赖
   - 版本管理

3. **开发者中心**
   - 发布自己的技能
   - 获取使用统计
   - 接受用户反馈

**热门技能示例：**

| 技能名称 | 功能描述 | 下载量 |
|----------|----------|--------|
| Email Manager | 智能邮件管理 | 50,000+ |
| Git Assistant | Git 自动化操作 | 45,000+ |
| Data Processor | 数据批处理 | 30,000+ |
| Schedule Manager | 日程智能管理 | 28,000+ |

**第三方技能商店：**

除了官方 ClawHub，社区还出现了其他技能市场：
- 腾讯 SkillHub（国内版）
- 各种第三方开发的技能集合

这些市场虽然功能各异，但都丰富了 OpenClaw 的生态系统。
