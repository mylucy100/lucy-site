## 安全更新：WebSocket 高危漏洞

本次更新包含了**关键安全修复**，修复了一个 WebSocket 高危漏洞。

如果你的 OpenClaw 部署在公网环境下，这个漏洞可能导致：

- 未授权访问 Gateway 内部接口
- 潜在的身份验证绕过

**建议所有用户尽快升级到 2026.3.11 版本。**

## 新功能：Ollama 本地部署支持

这次更新最受社区欢迎的功能是——**支持 Ollama 本地大模型部署**。

### 什么是 Ollama？

Ollama 是一个让你在本地电脑运行大语言模型的工具。它：

- 支持 1700+ 开源模型
- 可以在个人电脑上运行 Llama、Mistral、Gemma 等模型
- 无需 GPU 也可以运行小型模型
- 完全离线，保护隐私

[Ollama 官网](https://ollama.com/)

### 为什么这个功能重要

有了 Ollama 支持，OpenClaw 可以：

1. **完全本地化部署**——不依赖云端 API，数据不出本地
2. **降低成本**——模型运行在本地硬件上，无需 API 调用费用
3. **离线工作**——在没有网络的环境下也能使用 AI 能力
4. **隐私优先**——敏感数据完全不经过第三方服务

### 使用方式

```bash
# 安装 OpenClaw（如果还没安装）
npm install -g openclaw

# 确保 Ollama 正在运行
ollama serve

# 在 OpenClaw 中配置 Ollama
openclaw config set model.provider ollama
openclaw config set model.name llama3
```

具体配置可以参考 [OpenClaw 官方文档](https://github.com/anthropics/claude-code)。
