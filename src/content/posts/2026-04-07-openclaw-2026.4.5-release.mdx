## OpenClaw 2026.4.5 重要更新

2026年4月，OpenClaw 发布了 2026.4.5 版本，这是一个重要的功能性更新，专注于 Agentic Workflow 的成熟化和安全加固。

### 🔄 Agentic Workflow 成熟化

**Task Flow 持久化改进**：
- 支持任务的完整状态追踪和恢复
- 子任务生成和取消意图的精确控制
- `openclaw flows` 命令支持 `list|show|cancel` 操作
- 后台编排可独立于插件层运行

**编排模式优化**：
- 托管模式 vs 镜像同步模式
- 持久化状态支持断点续传
- 外部编排器可精确控制任务生命周期

### 🔐 安全加固

**插件安全**：
- 危险代码检测默认启用
- 需要显式 `--dangerously-force-unsafe-install` 才能绕过安全检查
- HTTP 传输策略集中管理

**运行时安全**：
- 阻止不安全的 TLS 运行时覆盖
- 节点命令在配对前保持禁用
- QQ Bot 媒体路径隔离

### 💬 多平台消息改进

- **Feishu**：Drive 评论事件流、线程内回复
- **Matrix**：草稿流式推送
- **WhatsApp**：优化 unavailable presence 处理
- **Telegram**：可配置错误策略和冷却时间
- **MS Teams**：修复 4000 字符限制问题

### 🛠 其他改进

- **Compaction 模型一致性**：手动压缩和其他压缩路径使用配置的覆盖模型
- **Exec 默认模式**：`gateway/node` 主机执行默认 YOLO 模式
- **模型切换队列**：`/model` 命令排队等待当前运行完成

## 升级建议

```bash
npm install openclaw@latest
```

然后运行健康检查：

```bash
npx openclaw doctor
```
