每个看过《钢铁侠》的人，都幻想过拥有一个贾维斯。

那个永远在线、无所不能、还能调侃你的 AI 管家。托尼·斯塔克一句「贾维斯」，它就出现了。

现实是，我们还没有核反应堆，也没有战甲。但是，一个 24 小时在线、能理解你、能帮你做事的 AI 助手——这个，现在真的可以有了。

今天我用 OpenClaw 来告诉你，如何无限接近贾维斯。

## 第一步：安装 OpenClaw

OpenClaw 是一个自托管的 AI 网关，把它理解成一个「消息中转站」——你从 WhatsApp、Telegram、Discord、飞书发消息给它，它帮你转给 AI，再把 AI 的回复发回来。

安装非常简单：

```bash
npm install -g openclaw@latest
```

然后运行引导配置：

```bash
openclaw onboard --install-daemon
```

这一步会安装 Gateway 服务，让它常驻后台。以后你不用管它，它就在那儿等着你的消息。

打开控制面板确认一切正常：

```bash
openclaw dashboard
```

本地默认地址是 `http://127.0.0.1:18789/`

到这里，基础设施就搭好了。就像贾维斯的「身体」已经就位，但还没有「大脑」。

## 第二步：配置一个聪明的大模型

贾维斯为什么强？因为它「懂」托尼。

它理解上下文、记住对话、知道托尼的习惯。这需要一个大容量的「大脑」。

OpenClaw 支持多种大模型提供商：

- **Claude**（推荐）— 推理能力最强，适合复杂任务
- **GPT-4o** — 综合能力强，响应快
- **DeepSeek** — 国产之光，性价比高
- **百炼/GLM** — 国内直连，不需要代理

配置方式很简单，在 `~/.openclaw/openclaw.json` 里设置：

```json
{
  "providers": {
    "anthropic": {
      "apiKey": "你的API密钥"
    }
  },
  "defaultAgent": {
    "model": "claude-sonnet-4-20250514"
  }
}
```

**我的建议**：如果预算允许，用 Claude Sonnet 4。它的推理能力和指令遵循能力是目前最强的，非常接近「理解你」的感觉。

就像给贾维斯装上了最好的 CPU。

## 第三步：装备合理的技能

贾维斯不只是聊天机器人。它能黑进电脑、控制战甲、分析数据。

在 OpenClaw 里，「技能」就是让 AI 具备实际操作能力的插件。

### 常用技能推荐

| 技能 | 功能 |
|------|------|
| **web_search** | 联网搜索，获取实时信息 |
| **email-imap-smtp** | 收发邮件 |
| **markdown-converter** | 文档格式转换 |
| **chrome-devtools** | 浏览器自动化调试 |
| **Docker** | 容器管理 |

安装技能：

```bash
# 通过 ClawHub 发现和安装技能
openclaw skill install <skill-name>
```

技能不在多，在于「合理」。

想想你最常做什么？写代码、查资料、发邮件、管服务器？按需配置。

**我的配置思路**：

1. **信息获取** — web_search（必装）
2. **日常办公** — email + markdown-converter
3. **开发运维** — Docker + chrome-devtools

这样你的贾维斯就有了「手」，不只是「嘴」。

## 第四步：优化记忆系统

贾维斯为什么能记住托尼说的每句话？因为它有记忆系统。

OpenClaw 的记忆系统由两部分组成：

### 1. MEMORY.md — 长期记忆

存放你的重要信息：项目、账号、习惯、偏好。

```markdown
# MEMORY.md

## 项目
- 项目A：/home/user/code/project-a
- 项目B：/home/user/code/project-b

## 偏好
- 编辑器：vim
- 时区：Asia/Shanghai
```

### 2. 每日笔记 — 短期记忆

`memory/YYYY-MM-DD.md` 记录当天发生的事。

### 记忆优化技巧

1. **定期整理** — 我有个定时任务，每天 22 点自动归档旧笔记
2. **主动更新** — 学到新东西、做了新决定，马上写进 MEMORY.md
3. **结构化存储** — 用标题、表格、列表，方便 AI 检索

好的记忆系统，让 AI 真正「认识你」。

就像贾维斯知道托尼喜欢咖啡不加糖。

## 第五步：多渠道接入

贾维斯无处不在——战甲里、家里、手机里。

OpenClaw 也支持多渠道接入：

- **Telegram** — 配置最简单，推荐新手
- **WhatsApp** — 全球最普及
- **Discord** — 游戏玩家首选
- **飞书** — 国内办公场景
- **iMessage** — 苹果生态

配置 Telegram 示例：

```json
{
  "channels": {
    "telegram": {
      "botToken": "你的机器人Token"
    }
  }
}
```

重启 Gateway：

```bash
openclaw gateway restart
```

现在，你可以在手机上随时随地跟你的贾维斯聊天了。

## 与真正贾维斯的距离

走到这一步，你已经拥有：

- ✅ 24 小时在线的 AI 助手
- ✅ 多渠道消息接入
- ✅ 聪明的大模型
- ✅ 实际操作的技能
- ✅ 能记住你的记忆系统

但是，我们离真正的贾维斯还有多远？

### 我们还没有的

| 能力 | 现状 |
|------|------|
| **完全自主** | AI 需要你发指令，不会主动行动 |
| **跨设备控制** | 不能直接控制智能家居、汽车 |
| **实时感知** | 不能「看」到你正在做什么 |
| **情感理解** | 只能分析文字，无法真正共情 |
| **自我进化** | 无法自主学习和升级 |

### 我们已经有的

| 能力 | 现状 |
|------|------|
| **随时响应** | ✅ 消息秒回 |
| **知识检索** | ✅ 联网搜索 + 本地记忆 |
| **任务执行** | ✅ 技能系统 |
| **多轮对话** | ✅ 上下文理解 |
| **个性化** | ✅ 记忆系统认识你 |

### 结论

我们大概达到了贾维斯的 **30%**。

这个 30% 已经非常有用——它能帮你查资料、写代码、发邮件、管服务器、记事情。

但剩下的 70%，需要的不只是技术突破，还有硬件生态的整合、安全边界的定义、以及 AI 自主性的伦理讨论。

**贾维斯是终极目标，OpenClaw 是现实的起点。**

从今天开始，打造你自己的 AI 管家吧。
